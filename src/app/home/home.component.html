Welcome!

Rational

Use this template for building a demo to gather help, to show a potential bug report or to set up a POC.

The template uses the latest available versions at the moment being:
  • angular: 4.3.5
  • bootstrap: 4.0.0-beta
  • zone.js: 0.8.16
  • reflect-metadata: 0.1.10
  • systemjs: 0.20.17
  • typescript: 2.4.2
  • rxjs: 5.4.3
  • plugins-typescript: 7.1.0 (typescript transpiler)
  • in-memory-web-api: 0.3.2 (used to intercept requests using the http module)

How to start

  1. Change the title in index.html
  2. Replace the favicon
  3. Update versions in package.json
  4. Install packages: npm install
  5. Run: npm start or firefox -P dev index.html (or any available browser)
  6. Add your packages
  7. Write your code!
